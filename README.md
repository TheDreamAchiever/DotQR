# DotQR ✦

A beautiful, fully working QR code generator that makes your QR codes 
actually look good. Customize dot styles, gradient colors, corner shapes, 
and even add your own logo — all in a single HTML file.

## Features

- Real scannable QR codes (works with any scanner)
- 5 dot styles — Dots, Rounded, Square, Classy, Pill
- 3 corner styles — Square, Round, Dot
- 8 gradient color themes
- Center logo / image upload
- PNG + SVG download
- Zero backend — runs in browser only
- Single HTML file — no installation needed

## Tech Used

- HTML / CSS / JavaScript
- [qr-code-styling](https://github.com/kozakdenys/qr-code-styling) library

## Usage

1. Clone the repo
2. Open `index.html` in browser
3. Enter URL, customize, download
